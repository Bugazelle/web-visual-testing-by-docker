## Hermione - Chrome - 模拟用户行为

Hermione模拟用户行为

> 注意: 
> 1. 假设把**samples**文件夹放入当前路径，并且将当前路径映射到docker容器中/tmp目录
> 2. 在.hermione.conf.js和docker启动命令中设有代理. 根据需要修改它或者移除它

1. 创建baseline

    ``` 
       docker run \
           -e VNC_RESOLUTION=1400x900 \
           -p 6901:6901 \
           -p 5901:5901 \
           -v $(pwd):/tmp \
           --privileged \
           -v /dev/shm:/dev/shm \
           --shm-size 2048m \
           bugazelle/web-visual-testing \
               /bin/bash -c "cd /tmp/samples/Hermione/Chrome/Hermione_SimulateUserActions; \
               export http_proxy=http://172.17.0.1:3128; \
               export https_proxy=http://172.17.0.1:3128; \
               npm install --save-dev chai; \
               unset http_proxy; \
               unset https_proxy; \
               selenium-standalone start & hermione --update-refs;"
    ```
